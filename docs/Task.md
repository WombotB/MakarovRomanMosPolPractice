# Описание задания по проектной практике

1.	Изучить синтаксис **Markdown** и вести всю документацию в этом формате.
2.	Создать статический веб-сайт (не менее 5 страниц), включающий: 
   -	Домашнюю страницу с аннотацией проекта.
  -	Страницу «О проекте» с подробным описанием.
  -	Страницу «Участники» с указанием личного вклада каждого.
  -	Страницу «Журнал» с минимум тремя постами о прогрессе.
  -	Страницу «Ресурсы» со ссылками на полезные материалы и партнёров.
3.	Оформить сайт с помощью *HTML и CSS*, обеспечить адаптивность и разместить на **GitHub**.
