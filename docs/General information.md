# Общая информация о проекте 

**Название проекта:**

«Виртуальная лаборатория»

**Цели и задачи проекта:**

Цель проекта – создать веб-приложение, реализующее функционал интерактивных лабораторных работ для учащихся средней и старшей школы. Проект направлен на то, чтобы сделать практическую часть учебного процесса доступной в любом месте и в любое время, не требуя дорогостоящего оборудования или специальных помещений.

**Задачи:**
1.	Провести анализ аналогов (*PhET Interactive Simulations, Labster, Яндекс Учебник*).
2.	Создать статический веб-сайт (*HTML/CSS*) для публичного представления проекта.
3.	Освоить инструменты **Git** и **Markdown**.
4.	Подготовить итоговый отчёт в репозитории **GitHub**.
