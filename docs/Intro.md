# ВВЕДЕНИЕ
Целью проектной практики являлась разработка веб-приложения «**Виртуальная лаборатория**», реализующего функционал интерактивных лабораторных работ для учащихся средней и старшей школы. Проект направлен на то, чтобы сделать практическую часть учебного процесса доступной в любом месте и в любое время, не требуя дорогостоящего оборудования или специальных помещений. С созданием веб-сайта для презентации результатов. В задачи практики входило:
-	изучение аналогов и лучших практик в области EdTech (от англ. Education technology — технологии образования);	
-	создание статического веб-сайта (HTML/CSS) для публичного представления проекта;
-	освоение инструментов Git и Markdown;
-	формирование итогового отчёта, готового для защиты. This is <u>underlined</u> text.


Практика проходила в период с <ins>03 февраля 2025 г. по 24 мая 2025 г.</ins> под руководством куратора от кафедры «Информатика и информационные технологии».
